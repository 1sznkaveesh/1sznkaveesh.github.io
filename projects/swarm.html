Swarm Robotics System

Active Deployment / Updates Ongoing

This project is focused on exploring decentralized coordination algorithms specifically tailored for smaller-scale swarm environments. Rather than heavily relying on a single complex control node, the objective is to let individual multi-agent units build grid consensus locally through peer communication networks.

By mapping simple interaction structures, the units can optimize path-routing and schedule adjustments layout dynamically without processing overhead spikes.
